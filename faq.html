FAQ

Since this should be an open source repository for beginners and new participants of the Hacktoberfest I thought it would be quite helpful to answer some questions here which everyone has who wants to beginn the journey of becoming a developer. It doesn't matter if you want to ask a question or if you want to answer a question or if you think it is not answered completely. Feel free to contribute.

Questions

  1. How to start?: I think it is a good idea to start with html and css to understand the basic concepts of how a website is brought to the screen of the user.
  2. Which ressources do I take?: There are many ressources on the internet. I recomment starting with a video course.
  3. Why doing open source?: Open Source is an awesome community. Give something back and celebrate it.
